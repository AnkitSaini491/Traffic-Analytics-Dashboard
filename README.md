# 🚦 Professional Traffic Analytics Dashboard

A professional **Traffic Analytics Dashboard** built using **Python, Pandas, and Matplotlib**. This dashboard analyzes traffic flow, vehicle distribution, congestion levels, average speed, truck movement, and road-wise traffic using KPI cards and interactive visualizations.

---

## 📊 Dashboard Features

- 🚗 Total Vehicles KPI
- 🚘 Cars KPI
- 🏍️ Bikes KPI
- 🚌 Buses KPI
- 🚚 Trucks Analysis
- ⚡ Average Speed KPI
- 🚦 Traffic Congestion KPI
- 📈 Monthly Traffic Trend
- 📊 Vehicle Type Analysis
- 🥧 Vehicle Share Distribution
- 🛣️ Road-wise Traffic Analysis
- ⏰ Peak Hour Traffic Analysis
- 🚗 Cars vs Bikes Comparison
- 🌙 Professional Dark Theme

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
